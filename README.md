## ✨ 项目简介

WuXueTab 是一款专为 Microsoft Edge 浏览器设计的轻量级新标签页管理扩展。它源于对现有商业化扩展的不满——既有安全风险又需付费订阅，因此我决定利用 AI 技术自主开发一个真正干净、免费且实用的替代方案。

## 🎯 核心特性

- **🛡️ 安全可靠**：完全开源，无任何病毒风险，隐私零泄露
- **💯 完全免费**：所有功能免费使用，无任何订阅或付费门槛
- **🚀 极速轻量**：代码精简，几乎不影响浏览器启动速度

- **⚙️ 高度可定制**：支持个性化配置，满足不同使用习惯

## 📦 安装方式

### 方法一：从 Microsoft Edge 加载扩展

1. 下载本扩展的源代码
2. 打开 Edge 浏览器，进入 `edge://extensions/`
3. 开启右上角的 **“开发人员模式”**
4. 点击 **“加载解压缩的扩展”**
5. 选择包含扩展文件的文件夹即可完成安装

### 方法二：使用 Edge 加载扩展包
- 直接加载已打包的 `.crx` 文件（如有提供）

## 🚀 快速开始

安装完成后：
1. 打开新的标签页即可看到 WuXueTab 界面
2. 根据个人喜好进行初步设置
3. 开始享受干净高效的上网体验
